# Calendly - Meeting Scheduling Platform

## Overview

A meeting scheduling and calendar coordination platform that eliminates back-and-forth emails when scheduling meetings. Users can share their availability, and invitees can book time slots that work for both parties.

## Key Features

- Personal availability management
- Meeting type configuration (1-on-1, group, round-robin)
- Calendar integration (Google Calendar, Outlook, etc.)
- Time zone detection and conversion
- Automated email notifications and reminders
- Conflict detection and prevention
- Buffer time between meetings
- Custom booking pages
- Rescheduling and cancellation
- Analytics and insights
